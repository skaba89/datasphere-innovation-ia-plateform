# Roadmap 90 jours

## Phase 1 - Fondation strategique et commerciale

Objectif : rendre le projet clair, vendable et presentable.

Livrables :

- vision strategique ;
- catalogue de services ;
- plaquette commerciale ;
- pitch deck ;
- landing page ;
- prompts agents ;
- workflow appels d offres ;
- modele de proposition commerciale.

## Phase 2 - Prospection et validation marche

Objectif : obtenir les premiers rendez-vous et tester les offres.

Actions :

- identifier 50 prospects ;
- preparer emails et messages LinkedIn ;
- analyser 3 appels d offres reels ;
- creer une demo de reponse augmentee par IA ;
- contacter ESN, cabinets, DSI, institutions et partenaires.

## Phase 3 - Premier chiffre d affaires

Objectif : signer les premieres missions.

Cibles :

- 1 a 3 missions de conseil ;
- 1 partenariat ESN ou cabinet ;
- 1 client pilote pour les agents IA ;
- 1 reference ou cas d usage public.

## Phase 4 - Industrialisation

Objectif : transformer la methode en systeme reutilisable.

Livrables :

- bibliotheque de prompts ;
- templates de livrables ;
- espace documentaire ;
- workflow qualite ;
- debut du MVP plateforme.

## Indicateurs de succes

- nombre de prospects qualifies ;
- nombre de rendez-vous ;
- nombre d offres envoyees ;
- taux de conversion ;
- chiffre d affaires signe ;
- nombre de livrables reutilisables ;
- qualite percue par les prospects.
