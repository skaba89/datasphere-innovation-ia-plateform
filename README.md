# DataSphere Innovation IA Plateform

Projet prive de construction d un cabinet de conseil augmente par IA, specialise en Data, IT, IA, transformation digitale et reponse aux appels d offres.

## Objectif

Creer une offre rentable et credible pour accompagner les entreprises et institutions en France et en Afrique, avec une priorite sur la Guinee.

## Methode

1. Formaliser les offres commerciales.
2. Structurer les agents specialises.
3. Creer les modeles de livrables.
4. Tester le marche avec des prospects.
5. Construire progressivement la plateforme.

## Regles

- Pas de secrets dans Git.
- Pas de fichier .env commite.
- Travail par branches apres l initialisation.
- Validation humaine obligatoire avant tout livrable client.
