# Operations commerciales

Ce dossier contient les outils simples de pilotage commercial de DataSphere Innovation.

## Objectif

Suivre les prospects, appels d offres, partenaires et opportunites afin de rendre le projet rapidement rentable.

## Fichiers

- crm-pipeline.md : definition du pipeline commercial.
- prospects-template.csv : modele simple de suivi des prospects et opportunites.
- opportunity-scorecard.md : grille d evaluation d une opportunite.
- prospecting-process.md : processus de prospection France, Guinee et Afrique francophone.

## Principe

Avant de construire une plateforme SaaS complete, DataSphere Innovation doit valider les offres, obtenir des rendez-vous, produire des propositions et signer les premieres missions.

Le CRM peut commencer sous forme de fichier CSV, Notion, Airtable, Google Sheet ou outil CRM leger. L important est de suivre les actions et les conversions.
