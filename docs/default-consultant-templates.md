# Templates consultants

## Objectif

Cette brique fournit une bibliotheque initiale de profils reutilisables pour accelerer les demonstrations et les premiers cas clients.

## Profils inclus

- Data Architect Senior
- Expert Reponse AO
- Consultant Data Gouvernance
- Business Analyst IT Data
- Expert Documentation Client

## Usage prevu

Ces templates servent de base pour creer rapidement des profils dans le catalogue.

Chaque profil contient un nom, un slug unique, un domaine, une seniorite, des langues, une description, un template d instructions, des outils cibles et des regles de gouvernance.

## Gouvernance

Les profils ne remplacent pas les consultants humains. Ils servent a accelerer l analyse, le cadrage, la documentation, la synthese et la preparation de livrables.

Toute sortie destinee a un client doit etre relue et validee par un humain responsable.
