# Offres de services

## Offre 1 - Pack Appels d Offres IA

Objectif : aider les entreprises et institutions a analyser, structurer et produire des reponses professionnelles aux appels d offres publics et prives.

Livrables :

- analyse du cahier des charges ;
- matrice de conformite ;
- strategie de reponse ;
- memoire technique ;
- memoire methodologique ;
- planning projet ;
- analyse des risques ;
- checklist de soumission.

## Offre 2 - Conseil Data, IT et IA

Objectif : accompagner les organisations dans leurs projets de transformation numerique.

Services :

- audit SI et data ;
- architecture data platform ;
- BI et dashboards ;
- data engineering ;
- cloud et DevOps ;
- IA generative ;
- gouvernance data ;
- documentation et conduite du changement.
