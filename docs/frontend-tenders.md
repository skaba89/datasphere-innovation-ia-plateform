# Frontend Appels d offres

## Objectif

Ajouter un espace frontend MVP pour gerer les appels d offres et leurs exigences directement depuis l interface web.

## Parcours cible

1. Connexion utilisateur.
2. Acces a l onglet Appels d offres.
3. Creation d un appel d offres lie a une opportunite existante.
4. Selection d un appel d offres.
5. Consultation des exigences associees.
6. Ajout d une exigence.
7. Visualisation rapide du statut des appels d offres.

## Limites MVP

Cette premiere version reste volontairement simple :

- pas encore d import automatique de cahier des charges ;
- pas encore d extraction IA ;
- pas encore de matrice de conformite avancee dans l interface ;
- pas encore d edition inline des exigences ;
- pas encore de workflow complet de validation.

## Prochaines evolutions

- detail appel d offres plus riche ;
- matrice Go No-Go graphique ;
- matrice de conformite ;
- upload document ;
- agent IA d analyse.
